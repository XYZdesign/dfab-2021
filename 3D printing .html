Ultimaker

It has a few function that we can make use of to adjust our workpiece

Function Infomation
Move: Able to move the workpiece around
Scale: Set it to the desired size we want
Rotate: to turn the work piece
Mirror: change the direction the workpiece is facing

After making it in fushion we than transfer it to Ultimaker


How to transfer to Ultimaker?

First Go to files > export


Under types, select STL files and save it to the folder you want


Open Ultimaker Cura 4.12.1 > files (red circle) > select the STL file


Put it to the size you want by using scale.


Before I slice it, I make sure to check my settings.

This are my settings

Setting Infomation
Layer height 0.25mm
Infill Density 10%
Infill Pattern Grid
Speed 80mm/s
Support Yes
Build Plate Adhesion Skirt

Going into preview mode to make sure everything is alright before I start printing
